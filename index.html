ABOUT ME

>

I've been obsessed with space since I was a kid, thanks to a healthy dose of sci-fi and a lot of daydreaming. Now, I actually get to work with spacecrafts at the European Space Agency, which is pretty cool. Still waiting on my own personal spaceflight, though.

>

I grew up in a small village in the outskirts of Venice, where the night sky was always amazing. That's probably where my interest in space really took hold. I studied Astronautical Engineering at the University of Padua, and now I'm in Darmstadt, Germany. I'm still looking up, just through a lot more screens and data.

>

When I'm not at work, I mess around with tech, do some coding to keep my brain busy, and read pretty much anything I can get my hands on. I also play a lot of roguelike games and run D&D for my friends. It's all about finding that balance between the technical stuff I love, and the creative outlets that keep me engaged.

some of my latest work

Spacecraft Data Analysis Platform

A custom Jupyter-based server which serves as a collaborative scripting and automation platform for Spacecraft Operators. Includes tools that automatically detect anomalies and fetch relevant data, prompting users with suggested recovery actions.

Space Ops Dashboard

Fetches data from several sources, including spacecraft telemetry, and stores it for efficient retrieval. The data is served to a Grafana server where users can build dashboards and set up alerts as they need.

Mission Operator toolbox

A website relying on a django backend that serves as a central hub for Spacecraft Controllers: helps with monitoring passes, aids with mission planning and anomaly recovery, performs data flow checks between ground systems and provides easy access to documentation.
