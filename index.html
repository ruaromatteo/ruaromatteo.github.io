some of my latest work

OpsHub

A custom Jupyter-based webserver which serves as a collaborative scripting and automation platform for spacecraft operators. Includes tools to automatically detect anomalies and fetch relevant data based on known signatures, prompting users with suggested recovery actions.

Phosphorite

A set of RPG game tools that simulate a fully configurable fictional spaceship, emulating a command shell, telemetry streams, comms, user sessions, LLM NPCs, and more, aimed at sci-fi TTRPG masters and players. Fully open-source.

Mission Operator toolbox

A website relying on a django backend that serves as a central hub for Spacecraft Controllers: helps with monitoring passes, aids with mission planning and anomaly recovery, performs data flow checks between ground systems and provides easy access to documentation on the fly.
